# Expense Tracker
Автор: Ибрагимова Зульфия Валидовна

## Описание
Программа для учета расходов с графическим интерфейсом. Реализована фильтрация по категориям, подсчет суммы и сохранение в JSON.

## Запуск
`python main.py`

## Список файлов
- main.py: основной код
- data.json: данные
- .gitignore: игнорируемые файлы

# Training Planner

**Автор:** Ибрагимова Зульфия Валидовна

## Описание

Training Planner — это GUI‑приложение для планирования и учёта тренировок. Программа позволяет:
* добавлять тренировки с указанием даты, типа и длительности;
* просматривать все записи в виде таблицы;
* фильтровать тренировки по типу и дате;
* сохранять данные в формате JSON между запусками приложения;
* загружать ранее сохранённые данные при запуске.

## Функциональность

* **Форма ввода**: поля для даты (формат ГГГГ‑ММ‑ДД), типа тренировки и длительности в минутах.
* **Валидация данных**: проверка корректности формата даты и положительности длительности.
* **Таблица**: отображение всех тренировок с колонками «Дата», «Тип», «Длительность».
* **Фильтрация**: возможность отфильтровать записи по типу тренировки (выпадающий список) и/или по конкретной дате.
* **Сохранение данных**: автоматическое сохранение в файл `trainings.json` при добавлении новой тренировки.
* **Загрузка данных**: при запуске приложение загружает ранее сохранённые тренировки из `trainings.json`.
